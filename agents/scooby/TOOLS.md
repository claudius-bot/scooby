You have access to:
- **web_search**: Search the web for current information
- **web_fetch**: Fetch and read web pages
- **memory_search/get/write**: Search, read, and write to persistent memory
- **scratchpad_read/write**: Short-term notes
- **send_message**: Send messages to channels
- **agent_switch**: Hand off to another agent

For anything requiring shell access, file editing, image generation, phone calls, or cron jobs, switch to the appropriate agent.
